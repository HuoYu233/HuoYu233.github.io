多重背包问题1

优化：二进制优化

for(int i=1;i<=n;i++){
    int a,b,s;
    cin>>a>>b>>s;
    //v w s;
    int k = 1;
    while(k<=s){
        cnt++;
        v[cnt] = a*k;
        w[cnt] = b*k;
        s-=k;
        k*=2;
    }
    if(s>0){
        cnt++;
        v[cnt] = a*s;
        w[cnt] = b*s;
    }
}

对物品进行二进制分组，组数为cntcntcnt，转化为01背包问题求解

n = cnt;
for(int i=1;i<=n;i++){
    for(int j=0;j<=m;j++){
        f[i][j] = f[i-1][j];
        if(j>=v[i]) f[i][j] = max(f[i][j],f[i-1][j-v[i]]+w[i]);
    }
}
cout<<f[n][m]<<endl;
