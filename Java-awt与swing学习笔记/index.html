布局管理器

  • 流式布局

    setLayout(new FlowLayout(FlowLayout.CENTER));

  • 东西南北中

    Button east = new Button(label);

    label：按钮名字

    frame.add(east,BorderLayout.East);

  • 表格布局

    Button but1 = new……

    frame.setLayout(new GridLayout(rows,cols));

    rows行cols列

    frame.add(but1);

    frame.pack() //自动选择最优秀的布局

总结

  • Frame是一个定义窗口

  • Panel无法单独显示，必须加到某一个容器中

  • 布局管理器：流式 东西南北中 表格（FlowLayout,BorderLayout,GridLayout）

  • 大小 定位 背景颜色 可见性 监听
