由数据范围反推时间复杂度及算法

2023-10-22 算法 391 字

一般竞赛时间限制在1s或2s，所以时间复杂度尽量控制在107−10810^7 - 10^8107−108

下面给出在不同数据范围下，代码的时间复杂度和算法如何选择：

  1. n≤30n \le 30n≤30，dfs+剪枝、状态压缩dp
  2. n≤100n \le 100n≤100，O(n3)O(n^3)O(n3)，Floyd、dp、高斯消元
  3. n≤1000n \le 1000n≤1000，O(n2)O(n^2)O(n2)，dp、二分、朴素Dijsktra、朴素Prim、Bellman-Ford
  4. n≤10000n \le 10000n≤10000，O(nn)O(n \sqrt n)O(nn​)，块状链表、分块、莫队
  5. n≤105n \le 10^5n≤105，O(nlogn)O(nlogn)O(nlogn)，sort、线段树、树状数组、set/map、heap、拓扑排序、堆优化Dijkstra、堆优化Prim、Kruskal、spfa、二分、CDQ分治、整体二分、后缀数组
  6. n≤106n \le 10^6n≤106，O(n)O(n)O(n)，单调队列、hash、双指针、BFS、并查集、kmp、AC自动机；常数比较小的O(nlogn)O(nlogn)O(nlogn)，sort、树状数组、heap、dijkstra、prim
  7. n≤107n \le 10^7n≤107，O(n)O(n)O(n)，双指针扫描、Kmp、AC自动机、线性筛素数
  8. n≤109n \le 10^9n≤109，O(n)O(\sqrt n)O(n​)，判断质数
  9. n≤1018n \le 10^{18}n≤1018，O(nlogn)O(nlogn)O(nlogn)，最大公约数、快速幂、数位dp
  10. n≤101000n \le 10^{1000}n≤101000，O((logn)2)O((logn)^2)O((logn)2)，高精度加减乘除
