Zset

在set基础上，增加一个值用于排序

  • zadd [key] [score] [value]
  • zrange [key] [from] [to]
  • zrangebyscore [key] [min] [max]
  • zrevrange [key] [max] [min]

min-> -inf max-> +inf

  • zrem [key] [value]

  • zcard [key]

  • zcount [key] [min] [max]

有序的set，存储班级成绩，工资表排序

添加权重：普通消息1，特殊消息2

排行榜应用实现

geospatial

推算地理位置的信息

  • geoadd [key] [longitude] [latitude] [member]

    一般下载城市数据一键导入

  • geodist [key] [from] [to] ([unit])

    返回指定位置之间的距离,使用unit选择单位(m,km…)

  • geohash

    返回一个或者多个元素的hash表示（了解即可）

  • geopos [key] [city]

    获取key的经纬度值

  • georadius [key] [longitude] [latitude] [unit] ([withdist]/[withcoord]) ([count])

    以某经纬度为中心，找到给定半径内的元素

  • geoeadiusbymember

    以成员为中心，找到给定半径内的元素（与georadius类似）

底层是Zset，可以用zset操作

hyperloglog

基数：不重复的元素

hyperloglog用于基数统计

优点：占用内存固定

  • pfadd [key] [v1] [v2] [v3] [v4]
  • pfcount [key]
  • pfmerge [destkey] [source1] [source2]
