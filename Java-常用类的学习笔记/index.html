Java 常用类的学习笔记

HawYior
2022-12-06
Java

常用类

Object类

超类，基类，位于继承树的最顶层

  • 作为参数，可接受任何对象
  • 作为返回值，可返回任何对象

getClass()方法

返回引用中储存的实际对象类型

hashCode()方法

  • 返回该对象的哈希码值

  • 哈希值根据对象的地址或字符串或数字使用hash算法计算出来的int类型的数值

  • 一般情况下相同对象返回相同哈希码

toString方法

  • 返回对象的字符串表示
  • 可以根据程序需求重写覆盖该方法

equals()方法

比较两个对象地址是否相等

一般重写

finalize()方法

  • 当对象被判定为垃圾对象时，由JVM自动调用此方法，用以标记垃圾对象，进入回收队列
  • 垃圾对象：没有有效引用指向此对象时，为垃圾对象
  • 垃圾回收：由GC销毁垃圾对象，释放数据储存空间
  • 自动回收机制：JVM的内存耗尽，一次性回收所有垃圾对象
  • 手动回收机制：使用System.gc()，通知JVM执行垃圾回收

从java9开始被弃用

包装类

基本数据类型所对应的引用数据类型

  • Object可统一所有数据，包装类的默认值是null
基本数据类型 包装类型
byte Byte
short Short
int Integer
long Long
float Float
double Double
boolean Boolean
char Character

类型转换与装箱、拆箱

基本类型 -> 栈

引用类型 -> 堆

基本类型 -> 引用类型：装箱

引用类型 -> 基本类型：拆箱

  • Number类

引用类型—>基本类型

  • 构造方法/valueof()

基本类型—>引用类型

JDK1.5之后，提供自动装箱和拆箱

基本类型和字符串的转换

  • 基本类型转换成字符串
  • 字符串转换基本类型
  • boolean字符串形式转换成基本类型

“true” —–> true

非”true” —–> false

Integer缓冲区

Java预先创建了256个常用的整数包装类型对象

在实际应用中，对已创建的对象进行复用

自动装箱：Integer.valueOf(num)

如果-128<num<127 使用valueOf(num)

如果不是, new Integer(num);

String类

  • 字符串是常量，创建后不可改变
  • 字符串字面值储存在字符串池中，可以共享

new String();创建两个对象 ，堆池各一个，可能浪费空间

常用方法

  • public int length()

返回字符串的长度

  • public char charAt(int index)

返回某个位置的字符

  • public boolean contains(String str)

判断是否包含某个字字符串

  • public char[] toCharArray()

将字符串转换成数组

  • public int indexOf(String str)

返回字符串首次出现的位置

  • public int lastIndexOf(String str)

返回字符串最后一次出现的位置

  • public String trim()

去掉字符串前后的空格

  • public String toUpperCase()

小写转成大写/大写转成小写(Lower)

  • public boolean endsWith(String str)

判断字符串是否以str结尾/开头(starts)

  • public String replace(char oldChar,char newChar)

将旧字符串替换成新字符串

  • public String[] spilit(String str)

根据str做拆分

  • 比较

  • equals() / equalsIgnoreCase()

比较字符串内容

  • compareTo()

比较字典序

可变字符串

  • StringBuffer

可变长字符串，JDK1.0提供，运行效率慢，线程安全

(1) append();追加

(2)insert();添加

(3)replace();替换

(4)delete();删除

  • StringBuilder

功能和StirngBuffer类似

可变长字符串，JDK5.0提供，运行效率快，线程不安全

  • 两者(1)效率比String高 (2)比String节省内存

BigDecimal类

double是近似值储存，运算可能会有精度损失

  • 加法：add()
  • 减法：subtract()
  • 乘法：multiply()
  • 除法：divide()

△ 除法中，bd1.divide(bd2)除不尽会报错

此时可以用

scale：指定精确到小数点后几位

mode：指定小数部分的取舍模式。

四舍五入取值为BigDecimal.ROUND_HALF_UP

Date类

表示特定的瞬间，精确到毫秒

大部分方法已被Calendar取代

Calendar类

  • Calendar提供了获取或设置各种日历字段的方法
  • 构造方法 protected Calendar()

由于修饰符是protected,所以无法直接创建该对象

SimpleDateFormat类

是一个以与语言环境有关的方式来格式化和解析日期的具体类

进行格式化（日期→文本）

解析（文本→日期）

System类

主要用于获取系统的属性数据和其他操作

构造方法私有

  • arraycopy(src,srcPos,dest,destPos,length)

src:源数组

srcPos:从哪个位置开始复制

dest:目标数组

destPos:从哪个位置开始放

length:复制长度

  • currentTimeMillis()

返回从1970-1-1到现在的毫秒数

可用于计算程序运行的时间
